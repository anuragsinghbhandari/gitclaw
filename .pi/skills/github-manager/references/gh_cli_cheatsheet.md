# GitHub CLI Cheatsheet

## Issues
- `gh issue list`: List issues in the current repository.
- `gh issue view <number>`: View details of an issue.
- `gh issue create --title "Title" --body "Body"`: Create a new issue.
- `gh issue comment <number> --body "Comment"`: Add a comment to an issue.
- `gh issue edit <number> --add-label "bug"`: Add a label to an issue.

## Pull Requests
- `gh pr list`: List pull requests.
- `gh pr view <number>`: View details of a PR.
- `gh pr checkout <number>`: Check out a PR locally.
- `gh pr create --title "Title" --body "Body"`: Create a new PR.

## Repository
- `gh repo view`: View the repository README and description.
- `gh repo clone <repo>`: Clone a repository.

## Search
- `gh search issues <query>`: Search for issues.
- `gh search prs <query>`: Search for pull requests.
